# 🍳 CyberChef: The Basics

CyberChef is a simple, intuitive web-based application designed to assist with a wide range of cybersecurity-related data manipulation tasks directly within your browser.

Think of CyberChef as a **Swiss Army knife for data**, containing hundreds of operations that can perform everything from simple encoding and decoding to advanced cryptographic functions.

Examples include:

* XOR Encoding/Decoding
* Base64 Encoding/Decoding
* AES Encryption
* RSA Decryption
* Data Extraction
* Format Conversion

CyberChef works using **recipes**, which are sequences of operations executed in a specific order against supplied input data.

## 🧭 Task 3: Navigating the Interface

CyberChef consists of four primary sections:

1. Operations
2. Recipe
3. Input
4. Output

### ⚙️ The Operations Area

The Operations Area contains all available CyberChef functions.

Operations are organized into categories and can be searched using the built-in search feature.

### Common Operations

| Operation | Description | Example |
|------------|-------------|----------|
| From Morse Code | Converts Morse code into text | `- .... .-. . .- - ...` → `THREATS` |
| URL Encode | Converts special characters into URL-safe format | `https://tryhackme.com` → Encoded URL |
| To Base64 | Encodes data into Base64 | `This is fun!` → `VGhpcyBpcyBmdW4h` |
| To Hex | Converts text into hexadecimal | `Hello` → Hex values |
| To Decimal | Converts text into decimal values | ASCII decimal conversion |
| ROT13 | Caesar cipher rotation of 13 characters | Text transformed by ROT13 |

### Helpful Feature

Hovering over an operation displays:

* Description
* Example
* Documentation links

### 🧪 The Recipe Area

The Recipe Area is considered the **heart of CyberChef**.

This is where operations are:

* Added
* Ordered
* Configured
* Executed

Users can drag operations into the recipe and modify their parameters.

### Features

#### Save Recipe

Allows recipes to be saved for future use.

#### Load Recipe

Loads previously saved recipes.

#### Clear Recipe

Removes all operations from the current recipe.

### BAKE!

The **BAKE!** button executes the recipe against the provided input.

### Auto Bake

When enabled:

```text
Recipes execute automatically whenever data changes.
```

No manual clicking of **BAKE!** is required.

### 📥 Input Area

The Input Area is where data is provided to CyberChef.

Input can be supplied through:

* Typing
* Copy/Paste
* File Upload
* Folder Upload

### Available Features

#### Add New Input Tab

Allows multiple inputs to be worked on simultaneously.

#### Open Folder as Input

Imports an entire folder.

#### Open File as Input

Imports a single file.

#### Clear Input and Output

Removes current data and results.

#### Reset Pane Layout

Restores the default interface layout.

### 📤 Output Area

The Output Area displays the results produced by the selected recipe.

### Available Features

#### Save Output to File

Exports results into a `.dat` file.

#### Copy Raw Output

Copies output directly to the clipboard.

#### Replace Input with Output

Uses the generated output as the new input.

#### Maximize Output Pane

Expands the output window.

## 🧭 Task 4: Before Anything Else

Before using CyberChef, it helps to follow a structured process.

The workflow consists of four steps.

### 1️⃣ Define the Objective

Ask yourself:

```text
What do I want to accomplish?
```

Example:

```text
I found a strange encoded string and want to reveal its hidden message.
```

### 2️⃣ Provide the Data

Paste or upload the data into the Input Area.

Examples:

* Encoded text
* Log files
* Hashes
* URLs

### 3️⃣ Select Operations

Determine which operations may be useful.

For example, if the data appears encoded, you may try:

* Base64
* Base85
* ROT13
* ROT47

Choosing the correct operation often requires research and experimentation.

### 4️⃣ Evaluate the Output

Ask yourself:

```text
Did I achieve my objective?
```

If not:

```text
Repeat the process.
```

Try different operations and approaches.

### Extract Email Addresses


### Extract URLs

Extracts URLs from text.

Example:

```text

```

### 🕒 Date and Time

#### Common Operations

| Operation | Description |
|------------|-------------|
| From UNIX Timestamp | Converts UNIX timestamp into readable date/time |
| To UNIX Timestamp | Converts date/time into UNIX timestamp |

#### UNIX Timestamp

A UNIX timestamp represents:

```text
The number of seconds since January 1, 1970 UTC
```

Example:

```text
Fri Sep 6 20:30:22 +04 2024
```

becomes:

```text
1725654622
```

### 📦 Data Format Operations

#### Common Operations

| Operation | Description | Example |
|------------|-------------|----------|
| From Base64 | Decodes Base64 | `V2VsY29tZSB0byB0cnloYWNrbWUh` → `Welcome to tryhackme!` |
| URL Decode | Decodes URL-encoded text | Encoded URL → Normal URL |
| From Base85 | Decodes Base85 data | `BOu!rD]j7BEbo7` → `hello world` |
| From Base58 | Decodes Base58 strings | `AXLU7qR` → `Thm58` |
| To Base62 | Encodes data into Base62 | `Thm62` → `6NiRkOY` |

### 🧠 Understanding Base64 Encoding

#### Step 1: Convert to Binary

The word:

```text
THM
```

ASCII values:

| Character | Binary |
|------------|---------|
| T | 01010100 |
| H | 01001000 |
| M | 01001101 |

Combined:

```text
010101000100100001001101
```

#### Step 2: Split into 6-bit Groups

```text
010101 000100 100001 001101
```

Convert to decimal:

| Binary | Decimal |
|----------|----------|
| 010101 | 21 |
| 000100 | 4 |
| 100001 | 33 |
| 001101 | 13 |

#### Step 3: Convert Using Base64 Index Table

| Decimal | Character |
|----------|-----------|
| 21 | V |
| 4 | E |
| 33 | h |
| 13 | N |

Result:

```text
VEhN
```

### 🌐 URL Encoding Quick Reference

| Character | UTF-8 Encoding |
|------------|---------------|
| : | %3A |
| / | %2F |
| . | %2E |
| = | %3D |
| # | %23 |

## 📝 Key Takeaways

* CyberChef functions as a cybersecurity Swiss Army knife.
* Recipes are collections of operations executed sequentially.
* Operations can encode, decode, encrypt, decrypt, convert, and extract data.
* The Recipe Area is the core of CyberChef.
* Auto Bake automatically processes data.
* Extractor operations help identify IP addresses, emails, and URLs.
* Date and Time operations convert UNIX timestamps.
* Base64 encoding converts binary data into text representation.
* URL encoding converts special characters into URL-safe format.
* CyberChef greatly simplifies common forensic, penetration testing, and incident response tasks.
